# Pixdrop

Pixdrop is a premium two-sided marketplace connecting customers with professional photographers.

## Features

- **Browse Photographers**: Filter by genre (Wedding, Portrait, Fashion, etc).
- **View Portfolios**: High-quality image galleries for each creative.
- **Book Instantly**: Seamless booking flow with customizable preferences.
- **Authentication**: User accounts for personalized experiences.

## Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS
- **Language**: TypeScript
- **Deployment**: Vercel

## Deployment Note

This project uses Next.js 15. Ensure your deployment environment supports Node.js 18+.
